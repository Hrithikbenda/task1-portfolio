# Personal Portfolio Website

## 📌 Overview
This project is a responsive personal portfolio website developed as part of the ApexPlanet Full Stack Development Internship.

The website showcases my profile, skills, projects, and contact information in a clean and modern layout.

## 🚀 Features

- Responsive design for different screen sizes
- Professional hero section
- About Me section
- Skills showcase
- Project portfolio section
- Contact section
- Interactive hover effects and animations
- Mobile-friendly navigation

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
